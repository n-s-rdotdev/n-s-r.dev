## Note

- If you are using drizzle, make sure it is installed and configured.
- If `Drizzle` is not yet configured, try [Drizzle Setup](https://n-s-r.dev/components/drizzle-setup) for quick installation.

## Installation

<CodeTabs>

<TabsListInstallType />

<TabsContent value="cli">
<Steps>

<Step> Add the complete Better Auth setup.</Step>

```bash
npx shadcn@latest add https://n-s-r.dev/r/better-auth.json
```

<Step>Create a .env file in the root of your project and add your database connection variable</Step>

> You may use `openssl rand -base64 32` to generate a new secret.

```bash
BETTER_AUTH_SECRET=
BETTER_AUTH_URL=http://localhost:3000 # Base URL of your app
```

<Step>Create Database Tables</Step>

```bash
npx auth@latest generate
```

</Steps>

</TabsContent>
