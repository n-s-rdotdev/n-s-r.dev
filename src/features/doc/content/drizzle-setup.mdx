## Installation

<CodeTabs>

<TabsListInstallType />

<TabsContent value="cli">
<Steps>

<Step> Add the complete Drizzle setup.</Step>

```bash
npx shadcn@latest add https://n-s-r.dev/r/drizzle.json
```

<Step>Create a .env file in the root of your project and add your database connection variable</Step>

```bash
DATABASE_URL= 
```

<Step>Applying changes to the database</Step>

```bash
npx drizzle-kit push
```

<Step>[Optional] If you are using `src` package, move `src/drizzle.config.ts` to `root`.</Step>

</Steps>

</TabsContent>
